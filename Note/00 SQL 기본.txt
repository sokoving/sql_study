### 정규형
- 반복적으로 적어야 하는 값은 분리해서 관리한다

### 기본키(Primary Key, PK)
- 각 행을 특정할 수 있는 한 개 이상의 칼럽
- 배열의 index 역할
- 고유하며 필수적인 값을 기본키로 한다(문자도 가능)

# 실습 데이터 모델
- #: 기본키
- *: 필수 속성
- o: 선택 속성


# 주요 데이터 유형 타입
CHAR(L), VARCHAR2(L) > String
NUMBER(L) > Int
NUMBER(L, D) > Double
DATE > LocalDate / LocalTime / LocalDateTime
